# 🏢 نظام رواتب الملز (AlMalaz-Payro)
**نظام إلكتروني لإدارة وحساب رواتب الموظفين باستخدام Python و Flask.**

## ✨ المميزات:
- ✅ قاعدة بيانات مدمجة (SQLite) لحفظ بيانات الموظفين.
- ✅ حساب صافي الرواتب تلقائياً.
- ✅ واجهة برمجية (API) لإضافة وعرض الموظفين.

## 🚀 تشغيل المشروع:
1. تثبيت المكتبات: `pip install flask flask-sqlalchemy`
2. تشغيل التطبيق: `python app.py`
